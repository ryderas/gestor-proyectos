¿Cómo funciona la colaboración? Los datos se guardan en tu propio Google Sheet. Tú y tu equipo abren este archivo HTML desde cualquier computadora y ven los cambios al instante. Nadie externo tiene acceso — solo quienes tengan el link de tu hoja.

Configurar Google Sheets (solo se hace una vez)

  1. Ve a sheets.google.com y crea una hoja nueva
  2. Menú: Extensiones → Apps Script
  3. Borra el código existente y pega el script de abajo
  4. Clic en Implementar → Nueva implementación
    Tipo: Aplicación web · Acceso: Cualquier persona
  5. Copia la URL del despliegue y pégala aquí abajo
Script para Apps Script:
function doGet(e){ var ss = SpreadsheetApp.getActiveSpreadsheet(); var sh = ss.getSheetByName('data') || ss.insertSheet('data'); var data = sh.getRange(1,1).getValue(); return ContentService.createTextOutput(data || '{}') .setMimeType(ContentService.MimeType.JSON); } function doPost(e){ var ss = SpreadsheetApp.getActiveSpreadsheet(); var sh = ss.getSheetByName('data') || ss.insertSheet('data'); sh.getRange(1,1).setValue(e.postData.contents); return ContentService.createTextOutput('ok'); }
